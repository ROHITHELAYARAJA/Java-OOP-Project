## 💻 Code Showcase

### 🧬 Inheritance + Encapsulation

```java
class Animal {
    private String name;

    Animal(String name){
        this.name = name;
    }

    public String getName(){
        return name;
    }

    public void makeSound(){
        System.out.println("Animal Sound");
    }
}
```

---

### 🎭 Runtime Polymorphism

```java
class Dog extends Animal {

    Dog(String name){
        super(name);
    }

    @Override
    public void makeSound(){
        System.out.println(getName() + " says Woof!");
    }
}
```

---

### 🚀 Execution

```java
Animal pet = new Dog("Bruno");
pet.makeSound();
```

Output:

```
Bruno says Woof!
```
